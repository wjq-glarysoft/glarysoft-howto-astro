What Are the Basic Windows System Repair Tools?

1\. Disk Cleanup This built-in utility helps free up space on your hard disk, improving your system’s performance by removing unnecessary files.

Steps to Use Disk Cleanup: - Click on the Start menu and type "Disk Cleanup" in the search bar. - Select the Disk Cleanup app from the list. - Choose the drive you want to clean (usually C:), then click OK. - The tool will calculate how much space you can free up. Check the boxes for the types of files you want to delete. - Click OK, then confirm by clicking Delete Files.

2\. Check Disk Utility (CHKDSK) CHKDSK examines your hard drive for errors and repairs them to prevent data loss.

Steps to Run CHKDSK: - Open File Explorer and go to This PC. - Right-click on the drive you want to check, usually C:, and select Properties. - Go to the Tools tab and click Check under Error checking. - Follow the prompts to either scan the drive or schedule a scan at the next startup if required.

3\. System File Checker (SFC) SFC scans and restores corrupted system files.

Steps to Run SFC: - Open Command Prompt as an administrator by typing "cmd" in the Start menu, right-clicking Command Prompt, and selecting Run as administrator. - Type the command: sfc /scannow and press Enter. - The scan will start, and it may take some time. Follow any instructions given if issues are found.

4\. Windows Update Troubleshooter This tool helps resolve problems related to Windows updates.

Steps to Use the Windows Update Troubleshooter: - Go to Settings by pressing Win + I. - Navigate to Update & Security, then Troubleshoot from the left panel. - Click on Additional troubleshooters, then select Windows Update, and click on Run the troubleshooter. - Follow the instructions provided to resolve any detected issues.
