Best Practices for File Naming Conventions

Use Clear, Descriptive Names Always use names that describe the file’s content or purpose. For example, instead of “doc1.docx,” use “2024-Budget-Report.docx.”

Avoid Special Characters and Spaces Windows can misinterpret certain characters in file names (like / \\ : \* ? " |). Replace spaces with hyphens or underscores to improve compatibility, e.g., “Family\_Photos\_2024” instead of “Family Photos 2024.”

Be Consistent with Date Formats Use a consistent date format that sorts well, such as YYYY-MM-DD (e.g., “2024-06-15\_Meeting-Notes.docx”). This helps keep files in chronological order.

Stick to Short, Manageable File Names Windows has path length limitations. Aim for concise names that avoid redundancy, e.g., “Invoice\_2024-06” instead of “June\_2024\_Invoice\_for\_Client\_Project.”

Version Control in File Names Indicate versions by adding “v1,” “v2,” or similar, before the file extension. For example, “Project-Proposal\_v2.docx.” This avoids overwriting and tracks changes.

Examples of Good File Names - Meeting\_Notes\_2024-06-15.docx - Resume\_John\_Doe\_v3.pdf - Invoice\_ClientX\_2024-06.xlsx

Organizing Files and Folders: Beginner Guide

How Should Beginners Set Up Their Folders?

Start with Broad Categories Create top-level folders for main areas of your work or personal life, such as “Documents,” “Photos,” “School,” “Work,” or “Finances.”

Organize Within Categories Inside each top-level folder, create subfolders for specific projects, clients, or time periods. For example, within “Photos,” add folders named by year and event: “2024\\Birthday\_Party.”

Use Templates for New Folders Set up a template folder structure. For example, every new client folder might have “Contracts,” “Invoices,” and “Correspondence” subfolders.

Windows Shortcuts and Libraries Take advantage of Windows Libraries (Documents, Pictures, etc.) and use shortcuts to quickly access frequently used folders.
