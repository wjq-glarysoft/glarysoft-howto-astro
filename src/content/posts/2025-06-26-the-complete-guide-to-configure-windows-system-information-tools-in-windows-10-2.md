Understanding what’s inside your computer is crucial for troubleshooting, upgrading, or simply keeping things running smoothly. Windows 10 comes with various built-in system information tools that help you view details about your hardware, software, and other components. This guide will walk beginners through how to access, configure, and make the most of these tools, along with tips for using third-party utilities like Glary Utilities for an even deeper look.

What Are Windows System Information Tools?

System information tools are built-in and external utilities that display details about your PC. These tools show you information such as processor type, memory size, device drivers, network settings, and more. Knowing how to use these tools helps you:

\- Diagnose hardware or software problems - Check compatibility for software or hardware upgrades - Find out if drivers are missing or outdated

Which Built-in Tools Should You Know About?

The main built-in Windows 10 system information tools include:

\- System Information (msinfo32) - Task Manager - Device Manager - Settings App (About section) - DirectX Diagnostic Tool (dxdiag) Let’s cover each tool, how to access it, and how to use it effectively.

How Do You Open and Use System Information (msinfo32)?

1\. Press the Windows key on your keyboard or click the Start menu. 2. Type “System Information” and select the app from the results. 3. The System Information window opens. You’ll see a summary of your system’s hardware, software, and components.

Practical Example: If you want to check how much RAM is installed, look at the "Installed Physical Memory (RAM)" entry under System Summary.

Tips for Beginners: - To export details, go to File > Export and save the report as a text file for sharing or record-keeping. - Use the search box at the bottom to quickly find items like "BIOS Version" or "Boot Device".

How Can You Use Task Manager for System Information?

1\. Right-click the taskbar at the bottom of your screen. 2. Click Task Manager. 3. Click the “Performance” tab.

Here, you can see details about CPU, Memory, Disk, Network, and GPU usage.

Practical Example: To monitor which applications are using the most memory, look in the "Processes" tab and sort by "Memory".

How Do You Access Device Manager?

1\. Press Windows + X and select Device Manager. 2. The Device Manager window lists all installed hardware components.

Practical Example: If your printer isn’t working, expand the “Printers” section to check for errors (yellow triangles).

How to Check System Information in the Settings App?

1\. Click the Start menu and select Settings (the gear icon). 2. Go to System > About. 3. Here you’ll find basic PC specs like device name, processor, installed RAM, device ID, product ID, and Windows edition/version.

How Do You Run the DirectX Diagnostic Tool (dxdiag)?

1\. Press Windows + R to open the Run dialog box. 2. Type dxdiag and press Enter. 3. The DirectX Diagnostic Tool will open, showing system, display, sound, and input information.

Practical Example: If you’re troubleshooting graphics issues or want to know your DirectX version, check the “Display” tab.
