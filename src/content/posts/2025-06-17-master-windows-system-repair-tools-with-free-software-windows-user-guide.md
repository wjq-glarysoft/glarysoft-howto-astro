What Essential Built-in Windows Tools Can You Use?

Windows offers several native repair and maintenance tools:

1\. System File Checker (SFC) SFC scans your system for corrupted or missing system files and automatically replaces them.

How to use: - Open Command Prompt as Administrator (Right-click Start > Command Prompt (Admin)). - Type sfc /scannow and press Enter. - Wait for the scan to finish. If errors are found, SFC will try to fix them.

2\. DISM (Deployment Image Servicing and Management) If SFC can’t fix an issue, DISM helps repair Windows images and the system file store.

How to use: - Open Command Prompt as Administrator. - Type DISM /Online /Cleanup-Image /RestoreHealth and press Enter. - Wait until the process completes—this might take a while.

3\. Disk Cleanup Disk Cleanup helps remove temporary files, system cache, and other junk files.

How to use: - Press Windows+S, type Disk Cleanup, and launch it. - Select the drive you want to clean. - Check the boxes for the file types to remove and click OK, then Delete Files.
