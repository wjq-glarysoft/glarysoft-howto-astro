What Are Windows System Tools?

Windows system tools are built-in utilities that help manage, maintain, and optimize your computer. These include Disk Cleanup, Task Manager, System Configuration, and more. Customizing these tools allows for better system management tailored to your preferences.

How to Customize System Tools?

1\. Disk Cleanup Customization - Open Disk Cleanup by typing "Disk Cleanup" in the Windows search bar and selecting it. - Select the drive you want to clean up, usually the C: drive, and click OK. - Once Disk Cleanup calculates the space you can free up, you'll see a list of file types. - Customize by checking the boxes next to the file types you wish to delete. Beginners should start with temporary files, recycle bin, and downloaded program files. - Click OK, then Delete Files to confirm.

2\. Task Manager Customization - Launch Task Manager by pressing Ctrl + Shift + Esc. - Click on the "Processes" tab to see running applications and background processes. - Customize by right-clicking on any column header to add or remove columns, such as "CPU Time" or "Network". - To manage startup programs, click the "Startup" tab. Disable unnecessary applications by right-clicking and selecting "Disable". This speeds up your system startup.

3\. System Configuration (msconfig) - Access System Configuration by typing "msconfig" into the Run dialog (Win + R) and pressing Enter. - Go to the "Boot" tab for boot options customization. Beginners can set "Safe Boot" if troubleshooting is needed. - Under the "Services" tab, check "Hide all Microsoft services" and disable third-party services that are not required. - The "Startup" tab will redirect to Task Manager for managing startup items as described above. - Click Apply, then OK, and restart your computer for changes to take effect.
