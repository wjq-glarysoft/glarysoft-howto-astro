Windows 10 includes several built-in diagnostic utilities that help you troubleshoot, optimize, and maintain your PC. These tools are especially valuable for beginners because they come pre-installed, are easy to use, and provide clear feedback about your system’s health. Understanding how to access and maintain these diagnostic utilities ensures your computer continues to run smoothly. This guide will walk you through the most important tools step by step, with real-world examples, and show you how to keep them effective.

What are Windows diagnostic utilities? Windows diagnostic utilities are tools built into the operating system that let you check for hardware and software problems, monitor system performance, and fix issues. They include programs like Check Disk, Memory Diagnostic, Performance Monitor, and Resource Monitor. These tools act as your PC’s "health check system," alerting you to potential problems before they become serious.

How to run Check Disk (chkdsk) to find drive errors Over time, your hard drive can develop errors that slow down your system. Windows has a tool called Check Disk to detect and correct these issues.

Step 1: Open File Explorer and select "This PC." Step 2: Right-click your main drive (usually C:) and select "Properties." Step 3: Go to the "Tools" tab and click "Check" under the Error checking section. Step 4: If errors are detected, Windows will guide you through fixing them.

Example: If your PC frequently freezes when opening files, running Check Disk can repair corrupted sectors on your hard drive, restoring smooth performance.

How to use Windows Memory Diagnostic to check RAM Faulty memory (RAM) often causes random restarts or Blue Screen errors. The Windows Memory Diagnostic tool helps identify these problems.

Step 1: Press Windows Key + R, type mdsched.exe, and press Enter. Step 2: Choose "Restart now and check for problems." Step 3: Your computer will reboot and automatically test your memory. Step 4: After the restart, Windows will show you the test results.

Example: If your PC crashes during gaming or heavy multitasking, running this diagnostic reveals whether the RAM is responsible.

How to view performance with Resource Monitor If your PC feels slow, you can use Resource Monitor to see what’s consuming your CPU, RAM, Disk, or Network.

Step 1: Right-click the taskbar and select "Task Manager." Step 2: Go to the "Performance" tab and click "Open Resource Monitor" at the bottom. Step 3: Browse the tabs for CPU, Memory, Disk, and Network to check which programs are using resources.

Example: If you notice your internet is slow, Resource Monitor can show you which app is using most of the bandwidth.

How to analyze long-term performance with Performance Monitor Performance Monitor gives detailed insights into how your PC behaves over time. This is useful if you want to track recurring issues.

Step 1: Press Windows Key + R, type perfmon, and press Enter. Step 2: In the left panel, expand "Data Collector Sets" and select "System." Step 3: Right-click "System Diagnostics" and choose "Start" to begin a report. Step 4: After a few minutes, review the report under "Reports > System > System Diagnostics."

Example: If your PC tends to slow down after hours of use, Performance Monitor will highlight whether the cause is CPU overheating, disk bottlenecks, or memory pressure.
