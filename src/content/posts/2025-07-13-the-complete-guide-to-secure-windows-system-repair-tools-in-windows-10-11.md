Section 1: For Beginners – Essential System Repair and Cleanup

1\. Using Windows Troubleshooters

Windows comes with automated troubleshooters for common issues.

Step-by-step: a. Click Start, then select Settings. b. Go to Update & Security > Troubleshoot. c. Choose the problem area (e.g., Windows Update, Internet Connections, etc.) and click “Run the troubleshooter.” d. Follow the on-screen instructions.

Example: If your printer isn’t working, select “Printer” and let Windows detect and fix the problem.

2\. System File Checker (SFC) – Repair Corrupted Files

Corrupted system files can cause crashes or errors. The SFC tool scans and repairs these files.

Step-by-step: a. Right-click the Start button and select “Windows Terminal (Admin)” or “Command Prompt (Admin).” b. Type: sfc /scannow c. Press Enter and wait for the process to complete. d. Follow any on-screen prompts to repair files.

Section 2: For Advanced Users – Deep Repair and Recovery

1\. Deployment Image Servicing and Management (DISM)

When SFC can’t fix system files, DISM is your next tool.

Step-by-step: a. Open Windows Terminal (Admin) or Command Prompt (Admin). b. Enter: DISM /Online /Cleanup-Image /RestoreHealth c. Press Enter and let the scan run (it may take a while). d. After completion, rerun SFC as described above.

2\. Safe Mode Repairs

Some repairs are easier in Safe Mode, which loads a minimal version of Windows.

Accessing Safe Mode: a. Press Windows Key + R, type msconfig, and hit Enter. b. Go to the Boot tab and check “Safe boot,” then restart. c. Perform repairs (e.g., uninstall problematic drivers or apps). d. Return to msconfig and uncheck “Safe boot” to boot normally.

3\. System Restore

If recent changes have destabilized your PC, System Restore can roll back your system to a previous working state.

Step-by-step: a. In the search box, type “Create a restore point” and open the System Properties window. b. Click “System Restore.” c. Choose a recent restore point and follow the on-screen instructions.
